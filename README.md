# Commit 규칙

> <br />
> - DailyAlgorithm 커밋은 사이트 - 문제이름과 같은 형태로 작성<br />
> - 문서의 상황을 알아보기 쉽게 아래와 같이 타입을 만든다<br />
> - `unfinish` - 작성을 다 하지 못하고 마무리하게 된 경우<br />
> - `finish` - `unfinish`한 내용을 마무리한 경우<br />
> - `update` - 문서 내용 보강 or 추가 <br /><br />
